Open Source · v1.0

GlobalShield
Gateway

Умный прокси-шлюз для локальной сети. Устанавливается одной командой на любое Linux-устройство.

bash <(curl -fsSL https://raw.githubusercontent.com/SanSanichSan/gsg/main/install.sh)

Требуются права root · Поддерживается Debian / Ubuntu / Armbian

Возможности

Что умеет GSG

🛡️

Smart-маршрутизация

Автоматически направляет заблокированный трафик через прокси, всё остальное — напрямую.

📱

Режим на устройство

Каждому устройству в сети — свой режим: Smart, Global, Bypass или Block.

🌐

Веб-интерфейс

Управление с любого браузера в локальной сети. Статистика трафика в реальном времени.

⚡

Прозрачный прокси

Работает на уровне nftables TPROXY. Клиентам не нужно настраивать прокси.

📦

Docker Compose

Все компоненты — изолированные контейнеры. Легко обновлять и откатывать.

🔄

Watchdog

Аппаратный watchdog перезагружает устройство при зависании системы.

Требования

Что нужно для установки

Linux ARM64 / x86_64 (Raspberry Pi, Orange Pi, мини-ПК)
Debian 11+ / Ubuntu 20.04+ / Armbian
Docker 20+ (установится автоматически)
Доступ root / sudo
Подписка в формате Clash / V2Ray / Sing-Box
Устройство должно быть шлюзом (gateway) сети
Установка

Пошаговая установка

1

Настройте устройство как шлюз

Устройство должно быть подключено к роутеру/провайдеру. В настройках роутера укажите IP устройства как основной шлюз (Default Gateway) для вашей сети, или включите DHCP на GSG и выключите на роутере.

2

Запустите установщик

Скопируйте команду и вставьте в терминал устройства (SSH или локально):

bash <(curl -fsSL https://raw.githubusercontent.com/SanSanichSan/gsg/main/install.sh)

Установщик спросит IP шлюза, LAN-интерфейс и диапазон DHCP.

3

Откройте веб-интерфейс

После сборки откройте в браузере:

http://<IP_УСТРОЙСТВА>:8080
4

Добавьте подписку

В разделе Подписка вставьте URL вашей прокси-подписки (Clash, V2Ray, Sing-Box). Нажмите Обновить — система скачает список узлов и применит конфиг.

5

Настройте устройства

В разделе Устройства выберите режим для каждого клиента сети. По умолчанию — Smart (умный обход блокировок).

Справочник

Режимы устройств

🧠 Smart

Блокированный трафик (YouTube, Instagram, Telegram и т.д.) идёт через прокси. Остальное — напрямую. Рекомендуется.

🌐 Global

Весь трафик через прокси-узел.

⚡ Bypass

Прямое подключение, прокси не используется. Удобно для умного дома, IoT-устройств.

🚫 Block

Полная блокировка сетевого доступа для устройства.

Архитектура

Компоненты системы

web-orchestrator

Веб-интерфейс

FastAPI + HTML dashboard. Порт 8080. Управление всеми настройками.

tunnel-provider

Прокси-ядро

Mihomo (Clash Meta). TPROXY на порту 12345. Загружает подписку и строит правила.

net-enforcer

Файрвол

nftables. Перенаправляет трафик на TPROXY. Управляет bypass-списком.

registry-dhcp

DHCP-сервер

dnsmasq. Раздаёт IP-адреса, регистрирует имена устройств.
